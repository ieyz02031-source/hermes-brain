**Hermes Brain** turns your AI agent's conversations, research, and notes into a durable, self-evolving knowledge graph. It discovers gaps, searches for missing knowledge, and fills them automatically ‚ÄĒ knowledge compounds like interest.

Based on [Andrej Karpathy's LLM Wiki pattern](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f).

## What it does

- **Self-evolution cycle** ‚ÄĒ automatically discovers knowledge gaps, searches for missing knowledge, creates notes
- **Semantic search** ‚ÄĒ vector similarity search using `all-MiniLM-L6-v2`
- **Knowledge graph** ‚ÄĒ automatically builds entity and relationship networks
- **Hot cache** ‚ÄĒ auto-updates recent context, millisecond response
- **Four note types** ‚ÄĒ entity, concept, exploration, diary
- **Reference validation** ‚ÄĒ checks broken links, isolated notes, duplicate titles
- **Cron automation** ‚ÄĒ runs self-evolution cycle daily
- **Local-first** ‚ÄĒ all data stored locally, no cloud upload

## Get started (30 seconds)

```bash
# 1 ‚ÄĒ Install
git clone https://github.com/ieyz02031-source/hermes-brain.git
cd hermes-brain
pip install sentence-transformers

# 2 ‚ÄĒ Build index
python scripts/semantic_index.py index

# 3 ‚ÄĒ Run self-evolution
python scripts/evolve.py run
```

No API keys required. Everything runs locally.
